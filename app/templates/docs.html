Documentation

Git Clone

git clone https://github.com/rahulkate173/Outreach-Engine-.git
cd Outreach-Engine-

Docker Installation

docker-compose up -d
# Server runs on http://localhost:8000

Manual Installation

python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python cli.py auth start

CLI Commands

smb02 auth start        # Start with authentication
smb02 server start      # Start FastAPI server
smb02 reset-model       # Reset cached model

API Endpoints

  • POST /api/auth/register - Register new user
  • POST /api/auth/login - Login user
  • POST /api/chat/message - Send chat message
  • GET /api/billing/quota - Get quota status
  • POST /api/billing/upgrade - Upgrade plan
